
1. Основной модуль rosreestr_online.py

	Программа содержит необходимые модули и тестово делает один запрос по адресу (на здание или участок по выбору)

2.Вспомогательные (прикладные):

	2.1.getCadNumBuilding.py

	Программа возвращает кадастровые номера земельных участков по адресу, убирая снятые с учета
	В результате создает таблицу в указанной папке и открывает ее.

	2.2. getCadNumParsel.py
	Программа возвращает кадастровые номера земельных участков по адресу, убирая снятые с учета
	В результате создает таблицу в указанной папке и открывает ее

	2.3. gettypeObjectFlat
	по адресу помещения, возвращает кадастровый номер
	Используется для наполнения ФИАС кадастровыми номерами

3. CertBundle.pem 
	Самописный сертификат. Иначе Росрееестр не отдает данные.

4. Описание сервиса прямого доступа
	Данные с сайта росреестра, по составлению запросов.
